# marked

It forked from [chjj/marked](https://github.com/chjj/marked) and I added TeX syntax support.

## TeX Syntax

You can use `$...$` for inline statements or `$...$` for block.

**inline**

```

```

**block**

```
$$

$$
```

## Options

**tex**

Type: `boolean`

Enable tex.
